# Contributing to the EB-2 NIW Educational Hub

Thank you for your interest in contributing. This repository is built on the Dhanasar framework and serves as a resource for EB-2 NIW candidates and professionals.

## How to Contribute

### Report Errors or Gaps

If you find an error, outdated information, or a gap in the guides:

1. Open an issue describing the problem
2. Include the specific guide or section
3. Suggest what information should be added or corrected

### Share Case Insights (Anonymized)

If you have insights from your own case (approved or denied), you can contribute anonymized case studies:

1. Write a brief summary (200-300 words) of your situation
2. Describe your case strength on each pillar
3. Share lessons learned or recommendations
4. Submit as a PR with filename `case-study-[your-initials].md`
5. All identifying information must be removed

### Improve Content

If you see a guide that could be clearer, more detailed, or better organized:

1. Fork the repository
2. Make your changes
3. Submit a pull request with a clear description of improvements

### Add Templates or Tools

If you've created a useful template, checklist, or tool for case analysis:

1. Document it clearly
2. Include instructions for use
3. Submit as a PR in the appropriate directory

## Standards for Contribution

- **Accuracy:** All claims must be grounded in Dhanasar or widely accepted immigration law sources
- **Neutrality:** Avoid promotional language or marketing tone
- **Clarity:** Write for EB-2 NIW candidates, not legal professionals
- **Citation:** Cite sources (government documents, research, case law) when making claims

## Process

1. **Fork** the repository
2. **Create a branch** for your changes
3. **Make your changes** following contribution standards
4. **Submit a PR** with clear description
5. **Review:** Maintainers will review within 2 weeks

## Questions?

If you have questions about contributing, open an issue or contact the repository maintainer.

Thank you for helping improve resources for EB-2 NIW candidates.
