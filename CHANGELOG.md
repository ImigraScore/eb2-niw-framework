# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-22

### Added
- **Complete Dhanasar Framework**: Comprehensive technical breakdown of Matter of Dhanasar, 26 I&N Dec. 884 (AAO 2016)
- **EB-2 Eligibility Analysis**: Advanced Degree Professional and Exceptional Ability pathways
- **Proposed Endeavor Construction**: Step-by-step guide for building defensible proposed endeavors
- **RFE Risk Signals**: Organized by Dhanasar prong with actionable mitigation strategies
- **Evidence Inventory Templates**: Checklists for substantial merit documentation
- **Case Strength Assessment Framework**: Decision trees for petition evaluation
- **FAQ Section**: 30+ real candidate questions with nuanced answers
- **Open Educational Resources**: CC BY 4.0 licensed, citable framework
- **GitHub Pages Integration**: Automated deployment and hosting
- **SEO Optimization**: Schema.org markup, meta tags, and rich results support

### Technical
- **Jekyll Static Site**: Markdown-to-HTML conversion with custom styling
- **Responsive Design**: Mobile-first approach with professional typography
- **Accessibility**: WCAG-compliant structure and navigation
- **Performance**: Optimized fonts, minimal CSS, fast loading
- **Version Control**: Git-based with contribution guidelines

### Documentation
- **Citation Standards**: APA 7th, BibTeX, and Zenodo DOI support
- **Source Attribution**: All claims grounded in primary legal authorities
- **Contributing Guidelines**: Standards for accuracy and contribution
- **License Documentation**: CC BY 4.0 with attribution requirements

### Compliance
- **USCIS Policy Manual**: Updated to January 15, 2025 guidance
- **Legal Disclaimers**: Clear separation from legal advice
- **Educational Focus**: Framework for candidates and professionals
- **Transparency**: Open-source with verifiable sources
